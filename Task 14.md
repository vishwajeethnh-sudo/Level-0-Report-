# Task 14: K Maps and Logic Gates
## Objective:
The main goal of this task was to understand the use of K Maps for simplifying boolean algebra. And to implement the simplified Boolean expression using digital logic circuits.
## Procedure:
Define the input and output variables for the system.
Door (A): Represents whether the door is open or closed.
Key (B): Represents whether the key button is pressed or not.
Alarm: Represents whether the alarm (LED/Buzzer) is activated.

Construct a truth table showing all possible combinations of door and key conditions and determine when the alarm should activate.
Plot the values from the truth table on a 2-variable Karnaugh Map.
Identify the groups of 1’s in the Karnaugh Map and derive the simplified Boolean expression.
Translate the simplified Boolean expression into a logic circuit using standard logic gates such as AND and NOT gates.
Implement the circuit using push buttons to represent the door and key inputs, and connect the output to an LED or buzzer.
Test the circuit by applying different input combinations and observe when the alarm activates.
